# v1.4.6 更新日志

这个版本主要是 BUG 修复版本，发布时间 `2020/05/27`

## 更新内容

BUG修复部分

- 修复`linux`系统下上传超过`5M`大小的文件报错问题
- 修复`Form::block`布局无法正常显示编辑数据问题[#156](https://github.com/jqhph/dcat-admin/issues/156)
- 修复数据表单弹窗重复刷新页面问题
- 修复`Form::selectResource`表单无法设置默认值问题

功能优化部分

- `ContextMenuActions`操作列宽度优化
- 移除已废弃的配置参数以及多余前端组件
